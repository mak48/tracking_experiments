# Регрессия зарплат HH.ru с нейросетью и MLflow трекингом

## Описание проекта
Предсказание зарплат по данным резюме с hh.ru с использованием полносвязной нейронной сети (FCN) и трекингом экспериментов в MLflow.

## Требования к заданию
- **MLflow URI**: http://kamnsv.com:55000
- **Название эксперимента**: LIne Regression HH
- **Название модели**: makoveeva_anastasia_fcn
- **Метрика качества**: r2_score_test

## Архитектура модели
- **Тип**: FCN (Fully Connected Network)
- **Слои**: 256 → 128 → 64 → 32 → 1
- **Активация**: ReLU
- **Регуляризация**: BatchNorm + Dropout (0.3)
